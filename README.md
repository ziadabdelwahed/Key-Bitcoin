# Phantom Key Hunter

BIP-39 multi-chain wallet scanner with automatic sweeping capability.

## Supported Chains
- BTC (Bitcoin)
- ETH (Ethereum)
- BSC (Binance Smart Chain)
- TRX (Tron)
- MATIC (Polygon)
- AVAX (Avalanche)
- FTM (Fantom)
- SOL (Solana)
